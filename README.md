## Installation

```bash
npm install
```

## Entwicklung starten

```bash
npm run dev
```

## Komponenten testen & dokumentieren

Storybook ist integriert, um Komponenten zu testen und zu dokumentieren.

```bash
npm run storybook
```

## Build

```bash
npm run build
```
